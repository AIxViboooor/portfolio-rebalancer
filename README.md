# Portfolio Rebalancer

A crypto portfolio rebalancing calculator with live prices from CoinGecko.

## Features

- **Search any coin** from CoinGecko's database
- **Live prices** in EUR or USD
- **Set target allocations** with sliders
- **See rebalancing actions** - exactly what to buy or sell
- **Visual allocation breakdown**

## How to use

1. Click "+ Add Asset" to search for coins
2. Enter how much of each coin you hold
3. Set your target allocation percentages (should total 100%)
4. See what you need to buy or sell to reach your targets
